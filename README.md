# 🃏 Pokémon Card Hover Effect

A simple and visually appealing front-end project that displays stacked Pokémon cards with an interactive hover effect. Each card is slightly rotated and layered using CSS to create a realistic deck-like UI. On hover, the selected card moves to the top, enhancing user interaction.

## 🚀 Features

* 🎴 Stacked card layout
* 🔄 Hover effect to bring card to front
* 🎨 Rotated card styling using CSS
* ⚡ Smooth and minimal UI
* 🖼️ Uses high-quality images

## 🛠️ Tech Stack

* HTML5
* CSS3

## 📂 Project Structure

my-project/
│── index.html
│── style.css
│── README.md

## ⚙️ How It Works

* Cards are positioned using **absolute positioning**
* Each card is rotated using **CSS transform**
* **z-index** is used to manage stacking order
* On hover, card gets higher z-index and comes to front
